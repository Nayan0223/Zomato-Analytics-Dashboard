# 🍽️ Zomato Analytics Dashboard

## Overview
This project analyzes restaurant data from Zomato using Tableau. The dashboard provides insights into restaurant ratings, cuisines, cities, online delivery availability, and customer preferences.

## Objectives
- Analyze restaurant performance.
- Identify top-rated restaurants.
- Compare ratings across cities.
- Understand cuisine popularity.
- Evaluate online delivery and table booking trends.

## Tools & Technologies
- Tableau
- Microsoft Excel
- Data Cleaning
- Data Visualization

## Dataset
Zomato Restaurant Dataset

## Key Insights
- Top-rated restaurants identified.
- Most popular cuisines analyzed.
- City-wise restaurant distribution visualized.
- Online delivery and table booking trends explored.

## Dashboard Features
- Interactive filters
- City-wise analysis
- Rating distribution
- Cuisine analysis
- Restaurant performance metrics
